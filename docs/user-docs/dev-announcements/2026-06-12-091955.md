**The workshop recording is live: Loop Engineering for Autonomous Agents**

Thursday's workshop is now up on YouTube.

We went deep on loop engineering — the shift from writing one-off prompts to designing loops that run your agents autonomously, on a remote machine, while you sleep.

In about 35 minutes:
- The 4-step playbook pattern behind every agent loop (read → work → verify → write)
- Keeping loops safe: max runs, stop signals, timeouts, and cost ceilings
- Loops vs schedules — when to reach for each
- Why remote loops on Trinity beat local ones that die when you close your laptop
- Live demos: PR review, docs, and multi-agent monitoring loops
